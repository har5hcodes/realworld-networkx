
The Jupyter Notebook is not included with Python, you will need to install Jupyter.
You can use pip to install Jupyter Notebook like this:

$ pip install jupyter

Starting the Jupyter Notebook Server
Now that you have Jupyter installed, 
you need to open up your terminal application and go to the folder 19EC30017_Assgn1
And run the following command:

$ jupyter notebook

Now, you can navigate to the respective question folder and check the code
